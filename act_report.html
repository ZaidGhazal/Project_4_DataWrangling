Data Investigation¶

After wrangling the three datasets created from the files (twitter-archive-enhanced.csv,image_predictions.tsv,tweet_json.txt), I analyzed the final dataset (twitter_archive_master.csv) to get some insights about the tweets by answering the following questions:

  • What is the most popular (liked) breed?
  • Which stage has the highest mean rating numerator?
  • Show the correlation between the rating_numerator, retweet_count, and favorite_count.
  • Visualize the sources usage frequency and find the one with the largest use.

Q1: What is the most popular (liked) breed?¶

The result is The most liked breed is: golden_retriever. This investigation done by grouping the data by the breed column and using NumPy and Pandas functions to find the favorite_count value and indicating the related breed.

Q2: Which stage has the highest mean rating numerator?¶

Using .groupby('dog_stage') function, I got each stage information including the rating numerator values. Consequently, I found the mean rate for each stage using Pandas .mean() and other functions and visualized the result using Matplotlib library.
